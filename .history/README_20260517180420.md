# The Merchant's Workshop

## Project Overview
**The Merchant's Workshop** is a console-based survival-horror inspired management game developed using C#. The project is heavily inspired by the weapon merchant and upgrade mechanics from the **Resident Evil** series, particularly the progression and customization systems found in **Resident Evil 4**. The application combines combat simulation, resource management, inventory systems, weapon customization, and progression mechanics into a fully interactive menu-driven console experience.

## Features

### 1. Weapon Upgrade System
One of the core systems of the project is the weapon upgrade system. The player can improve weapon statistics including damage, fire rate, reload speed, and ammo capacity using earned money. Upgrade costs increase progressively as weapons become stronger, preventing players from maxing out equipment too quickly and creating a natural progression curve. Once a weapon reaches its maximum upgrade level, the player can unlock exclusive upgrades that provide unique bonuses such as critical hit multipliers, armor-piercing rounds, or increased damage output. This system mimics progression mechanics commonly used in modern action and RPG games while reinforcing object-oriented programming concepts.

Each weapon belongs to a unique category such as handguns, shotguns, riles, SMGSs, or magnums, and every category behaves differently during gameplay. For example, shotguns focus on extremely high damage output, SMGs prioritize fire rate and ammo capacity, rifles provide balanced long-range performance, while magnums deliver devastating damage at a very high upgrade cost. This creates meaningful decision-making and encourages the player to experiment with different play styles and weapon builds.

#### What Each Weapon Contains

| Feature   |   Purpose  |
|-----------|------------|
| Name | Weapon Name |
| Weapon Type | Determines Scaling |
| Damage | Amount of Damage Dealt |
| Fire Rate | Shooting Speed |
| Capacity | Ammo Size |
| Reload Speed | Reload Time |
| Upgrade Level | Current Upgrade Progress |
| Attachments | Equipped Upgrades |
| Exclusive Upgrade | Final Special Bonus |
