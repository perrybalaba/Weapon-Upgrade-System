Weapon Upgrade System

## Project Overview
The Weapon Upgrade System is a console-based survival-horror inspired management game developed using C#. The project is heavily inspired by the weapon merchant and upgrade mechanics from the **Resident Evil** series, particularly the progression and customization systems found in **Resident Evil 4**. The application combines combat simulation, resource management, inventory systems, weapon customization, and progression mechanics into a fully interactive menu-driven console experience.
