# Weapon-Upgrade-System

## Project Overview
The Weapon Upgrade System is a console-based C# application inspired by the weapon merchant and upgrade mechanics found in Resident Evil 4. The project simulates a survival-horror style weapon shop where the player manages money, purchases weapons, upgrades weapon statistics, and customizes equipment strategically.

The application focuses heavily on object-oriented programming concepts by using multiple interacting classes such as `Weapon`, `Player`, and `Merchant`. Each weapon belongs to a specific category, such as handguns, rifles, shotguns, SMGs, or magnums, and each category scales differently when upgraded. The player must carefully decide how to spend limited currency to improve weapon performance and build an efficient inventory.
